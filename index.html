RSU Usage & Steps

The SmartBob Remote Start Unit (RSU) is an add-on for the BinMaster SmartBob system. It adds an external start input (dry contact), a 4–20 mA analog output for PLCs/DCS, and a remote display of level, distance to product, percent full, and bob status. Measurements and settings are kept in nonvolatile memory so they survive power loss.

Installation & signals (summary)

  • Power: 16 VAC (e.g. recommended transformer); follow local codes (manual suggests 14–18 AWG for that feed).
  • RS‑485: Quality shielded twisted pair to the SmartBob remote; polarity must be correct. Only one remote; SmartBob address must be 1 (factory default).
  • 4–20 mA loop: Max loop resistance 750 Ω. If exceeded (or loop open), the LOOP FAULT LED on the faceplate lights and the analog output is invalid.
  • Wiring diagram: See the Wiring section below and full SMARTBOB RSU Manual PDF.

Power-up behavior

  • If bin height is still 000.0, the RSU shows the splash (BINMASTER Smart Bob / Remote Start Unit / With 4–20 mA Output / Press any Key). Any key goes to the Main Menu.
  • If a bin height was already stored, the RSU powers up into Display Mode (operational screen), ready for external starts or timed measurements.

Main Menu

  • + → Setup Menu
  • − → Test Menu (4–20 mA loop test)
  • Enter → Display Mode (normal operation). Requires a valid bin height; height 0 produces a display error and you must configure height first.

Setup Menu

  • + → Set Bin Height (required for correct 4–20 mA scaling).
  • − → Configuration Menu (external start, units, interval timer).
  • Enter → back to Main Menu.

Bin height entry

  • Range 0.1 to 199.9 (feet per manual; metric path uses meters when selected).
  • + / − adjust the active digit; Enter accepts that digit and advances. After the tenths digit, the RSU asks if the value is correct: + confirms, − starts over.

Configuration Menu

  • + → External Start: Momentary (edge: one measurement per closure) vs Maintained (level: repeats while closed). Factory default: momentary. Enter returns to Setup Menu.
  • − → Units: English (feet) vs Metric (meters). Enter → Setup Menu.
  • Enter → Interval Timer menu.

Interval timer

  • + → set interval (1 minute steps up to 24 hours; enter digits with +/−/Enter per manual).
  • − → Enable / Disable timer (toggles).
  • Enter → Setup Menu.

Timer notes (manual): Each time the timer is enabled and you return to Display Mode, a measurement runs, even if you enabled then disabled the timer before leaving menus. With the timer off, remaining time is remembered and resumes when re-enabled.

Test Menu (4–20 mA)

  • From Main Menu, − opens the test menu for loop calibration/checking.
  • + → force 20 mA · − → force 4 mA (per menu labels on the RSU). Enter → Main Menu.
  • Optional for troubleshooting; the RSU does not require calibration for rated accuracy.

Display Mode (operation)

  • Normal run mode: accepts external start and runs interval measurements when the timer is enabled.
  • Shows bin %, product height, distance to product, bob line state, and optional countdown when the timer is on.
  • Any key exits Display Mode back to the Main Menu (manual).

Faults & errors

  • LOOP FAULT LED: Lit if the 4–20 mA loop is open or resistance > 750 Ω; output not valid.
  • Bob / measurement fault: If SmartBob senses inconsistent drop vs retract (> 6 in per manual), the RSU can show an error; the loop may go to 20 mA as a fault indication for the PLC.

  1. Locate the power transformer near the SBR/SBRX remote to limit voltage drop.

Troubleshooting

About this webpage

Browser demo only. Not wired to a real RSU or SmartBob.

SmartBob not responding / COMM‑style messages

You often won’t see a single short “COMM” flash. The display may step through lines such as remote did not respond, check RS‑485 wiring, or check address (wording varies by firmware). That’s one fault explained in pieces, not three separate failures.

If you see that kind of message:

  • Tighten RS‑485 terminals and check the pair for breaks or damage.
  • Match +/− at the RSU and the SmartBob.
  • Set the remote to address 1 (RSU talks to one SmartBob at factory default).
  • To narrow it down, disconnect RS‑485 at one end, then the other, and see whether the fault follows the cable or a box.
  • Use shielded twisted pair and land shields per the RSU manual wiring drawing.

Bin height zero / can’t use Display Mode

What it means: No valid bin height is stored.

What to do: Go into Setup and enter a real bin height.

LOOP FAULT LED lit

What it means: The 4–20 mA loop is open or loop resistance is above 750 Ω, so the analog output isn’t reliable.

What to do: Repair the loop before trusting the mA signal.

Override, filling, or bob stuck

What to do: Follow your SmartBob manual and your site’s safety rules.

Power, blank screen / dead keys

Weak or missing supply: Expect about 16 VAC at the RSU terminals per installation.

What the RSU expects: RS‑485 to one SmartBob remote at binary address 1, wired per the RSU manual (Figures 1–2 for SBR/SBRX and SBRII).

LCD blank with power OK, or keys don’t work: Likely bad faceplate electronics.

Replacing the RSU: The RSU unit is retired. Call the BinMaster Sales team and order a new C‑100 unit.
